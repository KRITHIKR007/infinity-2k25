Admin Token Login

Use this page as a last resort if other login methods fail.

Step 1: Run this in your browser console on a page where you're already logged in:

async function getToken() {
  const { data } = await supabase.auth.getSession();
  return data.session ? data.session.access_token : null;
}
console.log(await getToken());
