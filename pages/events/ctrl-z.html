CTRL+Z

A team-based debugging competition where participants fix scrambled code to match expected outputs.

Event Overview

CTRL+Z is an exciting team-based debugging competition that tests participants' problem-solving skills, coding knowledge, and ability to work under pressure. Teams will receive scrambled code with logical and syntax errors that they must identify, fix, and reorder to match the expected output.

This challenge simulates real-world debugging scenarios where programmers must understand and fix code written by others. It's an excellent opportunity for participants to showcase their analytical thinking, programming knowledge, and collaborative skills while competing in a fast-paced environment.

Rules & Guidelines

  • Team Size: 3-4 members per team.
  • Competition Structure: Each team will receive a set of scrambled code snippets containing various logical and syntax errors.
  • Teams must reorder, debug, and ensure the output matches the expected result.
  • Judging Criteria:
  • Accuracy (50%): Correctness of the fixed code and its output
  • Completion Rate (30%): Number of problems solved correctly
  • Efficiency (10%): Quality and cleanliness of the solution
  • Speed (10%): Time taken to complete the challenges
  • Teams will be provided with computers and necessary software.
  • Use of unauthorized resources, internet access, or communication with people outside your team is prohibited.
