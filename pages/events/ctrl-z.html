CTRL+Z

UNDO THE CHAOS

Venue

FET

Team Size

3-4 members

Registration Fee

Event Description

CTRL+Z is a team-based debugging competition where participants will receive scrambled codes containing logical and syntax errors. Your mission: identify and fix the issues to make the code work as intended.

Organized by Neuron, DATA.AI, Info-Sphere, and Cognito clubs, this event tests your problem-solving abilities, coding knowledge, and teamwork under pressure.

Competition Structure

Challenge Format

  • Each team will receive multiple code snippets with different types of errors
  • Expected outputs will be provided for each code snippet
  • Teams must debug the code to produce the correct output
  • Programming languages include Python, Java, JavaScript, and C++
  • Limited time to complete all debugging challenges

Error Types

Syntax Errors

Missing semicolons, brackets, typos, and other syntax issues

Logical Errors

Incorrect algorithms, faulty conditions, and flawed loop structures

Runtime Errors

Null pointer exceptions, division by zero, and array out of bounds

Optimization Issues

Inefficient code that needs to be improved for performance

Judging Criteria

Accuracy & Correctness (50%) - How many errors were fixed correctly

Completion Rate (30%) - Number of challenges completed

Efficiency (10%) - Quality and elegance of solutions

Speed (10%) - Time taken to complete the challenges
